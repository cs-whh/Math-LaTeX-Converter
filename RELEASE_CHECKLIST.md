## 打包步骤

1. 确保所有文件都在项目目录中
2. 在 Chrome 扩展管理页面点击"打包扩展程序"
3. 选择项目目录
4. 生成 .crx 文件和 .pem 密钥文件
5. 保存 .pem 文件用于后续更新

## 发布说明模板

```
# Math LaTeX Converter v1.0

## 新功能
- 支持 DeepSeek Chat 和 Grok 网站
- 自动转换 LaTeX 数学公式格式
- 适用于 Obsidian 等 Markdown 编辑器

## 使用方法
1. 安装扩展
2. 在支持的网站上复制包含数学公式的内容
3. 粘贴到 Markdown 编辑器中，格式会自动转换

## 支持的转换
- \( ... \) → $ ... $
- \[ ... \] → $$ ... $$
```
